   git init
   git add .
   git commit -m "Initial commit: Pailit MVP"

   • Connect your GitHub repo
   • Click Deploy
   • Done! Live in 30 seconds.

Result: Your app is live at https://pailit-[random].vercel.app

💡 HOW USERS EXPERIENCE IT:
═══════════════════════════════════════════════════════════════════════════════

1. Open app at https://pailit-yourname.vercel.app
2. Click "+ Tambah Tugas"
3. Create task: "Finish presentation", deadline "10:00", nominal "10.000"
4. Today's list shows task
5. At 10:00: notification pops up
6. Click "✓ Kelar" (complete) or "✗ Gagal" (fail)
7. If fail: balance goes RED, -Rp10.000
8. At 21:00: nightly report shows stats
9. Repeat daily, balance stays in red until you shape up 🔴
